# YouTube Downloader

## Overview
YouTube Downloader is a simple Python-based application that allows users to download videos from YouTube using a video URL.  
The project demonstrates basic concepts of Python scripting, user input handling, and file downloading automation.

This project was developed as a practice project to strengthen Python programming skills and understand how automation tools interact with web resources.

## Features
- Download YouTube videos using a video URL
- Simple and lightweight Python script
- Automatic saving of downloaded videos
- Easy command-line interaction

## Technologies Used
- Python
- pytube
- Command Line Interface (CLI)
- Git and GitHub

## Skills Demonstrated
- Python scripting
- Automation using external libraries
- Handling user input
- File downloading and management
- Basic project structuring

## Installation
1. Clone the repository
```bash
git clone https://github.com/richmonddeanlim/Youtube-Downloader.git
```
3. Navigate to the project directory
4. Install dependencies [ installing pytube and mmfpeg ]

## Usage
Run the Python script:
```bash
python YouTube.py
```
Then paste the YouTube video URL when prompted.
The video will be downloaded and saved locally on your device.
notes: pls change the directory location in the code.

## Example Workflow
1. Run the script
2. Paste a YouTube video link
3. The program processes the link
4. The video is downloaded automatically
